# CodeAlpha_FitnessTrackerApp

## Overview
Simple Java GUI app to track fitness data (steps, calories, workout).

## Features
- Input fitness data
- Store in SQLite database
- View saved records

## How to Run
1. Add sqlite-jdbc.jar in project folder
2. Compile:
   javac -cp ".;sqlite-jdbc.jar" Main.java
3. Run:
   java -cp ".;sqlite-jdbc.jar" Main

## Tech Used
- Java Swing
- SQLite
